L'objectif est de comprendre QUAND, POUR QUI et POURQUOI les sinistres surviennent.

Taux de Sinistralité (Claim Frequency) :

Calculer le nombre de sinistres par rapport au nombre de polices (ou à l'exposition). Analyser ce taux par :

Age et Gender de l'assuré.

Destination et Duration du voyage.

Product.Name (une assurance "premium" a-t-elle plus de sinistres ?).

Agency.Type ou Distribution.Channel (certains canaux vendent-ils à une clientèle plus risquée ?).

Analyse des Causes de Sinistres (via Claim.Status) :

Si Claim.Status contient des détails comme "Annulation", "Maladie", "Bagages", "Rapatriement" :

Identifier les causes principales de sinistres.

Étudier le coût moyen par type de sinistre.

Sévérité des Sinistres (Claim Severity) :

Si vous avez le montant des sinistres (Claim Amount), analysez le coût moyen par sinistre selon les mêmes variables (Age, Destination, Produit...).

Combinaison Fréquence * Sévérité = Coût Pur (Pure Premium). C'est la base du pricing.
